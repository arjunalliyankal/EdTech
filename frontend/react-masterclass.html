What's New in React 19?

React 19 brings powerful new features for building modern web applications, focusing on performance, developer experience, and seamless data handling.

1. React Server Components (RSC)

RSC allows you to render components on the server, sending only the necessary HTML and minimal JSON to the client. This reduces bundle size and improves initial load performance.

2. Actions (Server Actions)

Actions allow you to write functions that execute on the server, callable directly from your client components. This simplifies data mutation and form handling, often removing the need for separate API endpoints for simple tasks.

async function updateUser(formData) {
  'use server';
  await db.user.update({...});
}

3. useOptimistic Hook

The `useOptimistic` hook lets you show a different state while an async action is underway. This is perfect for "optimistic UI" updates where the interface reflects a change instantly, before the server confirms it.
